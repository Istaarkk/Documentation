# Pwn Environment

> Notes d'installation et de configuration pour un environnement de travail **pwn** propre et fonctionnel.

---

## Pwntools

### Installation
```bash
pip install pwn
pip install pwntools
```

---

## Pwninit

### Installation
```bash
sudo apt update
sudo apt install -y cargo

cargo install pwninit
~/.cargo/bin/pwninit
echo 'export PATH="$HOME/.cargo/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
pwninit --version
```

---

##  Pwndbg

### Installation
```bash
git clone https://github.com/pwndbg/pwndbg
cd pwndbg
./setup.sh
~/.gdbinit
```

---

## ASLR

### Désactiver
```bash
echo 0 | sudo tee /proc/sys/kernel/randomize_va_space
```

### Réactiver
```bash
echo 2 | sudo tee /proc/sys/kernel/randomize_va_space
```
